## 2. 보안 설정

1. 접근제어 설정

```

```

2. 패스워드 복잡도 설정

```
vi /etc/security/pwquality.conf
lcredit, dccredit, ocredit = 1 minlen=8
```

3. 패스워드 복잡도 및 ROOT 계정의 원격 접속 제한

```
vi /etc/pam.d/system-auth
vi /etc/pam.d/password-auth

auth required pam_tally2.so deny=5 unlock_time=1800
account required pam_tally2.so
password    requisite   pam_cracklib.so retry=3 dcredit=-1 lcredit=-1 ocredit=-1
password sufficient pam_unix.so remember=2 추가
```

4. 계정관련 설정(암호, 기간, umask)

```
/etc/login.defs
PASS_MIN_LEN 8
PASS_MAX_DAYS 90
PASS_MIN_DAYS 7
umask 022
```

## 2-1. 보안 설정 ansible 코드

- hosts: linux
  tasks:
    - name: Create /etc/hosts.allow file
      lineinfile:
        path: /etc/hosts.allow
        line: 'sshd: {IP_ADDRESS}'

    - name: Create /etc/hosts.deny file
      lineinfile:
        path: /etc/hosts.deny
        line: 'ALL: ALL'
        create: yes

    - name: Update /etc/security/pwquality.conf file
      lineinfile:
        path: /etc/security/pwquality.conf
        regexp: "{{ item.regexp }}"
        line: "{{ item.line }}"
      loop:
        - { regexp: '^lcredit', line: 'lcredit = 1' }
        - { regexp: '^dcredit', line: 'dcredit = 1' }
        - { regexp: '^ocredit', line: 'ocredit = 1' }
        - { regexp: '^minlen', line: 'minlen = 8' }

    - name: Update /etc/login.defs file
      lineinfile:
        path: /etc/login.defs
        regexp: "{{ item.regexp }}"
        line: "{{ item.line }}"
      loop:
        - { regexp: '^PASS_MIN_LEN', line: 'PASS_MIN_LEN 8' }
        - { regexp: '^PASS_MAX_DAYS', line: 'PASS_MAX_DAYS 90' }
        - { regexp: '^PASS_MIN_DAYS', line: 'PASS_MIN_DAYS 7' }
        - { regexp: '^umask', line: 'umask 022' }

    - name: Set file permissions
      file:
        path: "{{ item.path }}"
        mode: "{{ item.mode }}"
      loop:
        - { path: '/etc/passwd', mode: '644' }
        - { path: '/etc/group', mode: '644' }
        - { path: '/etc/hosts', mode: '644' }
        - { path: '/etc/services', mode: '644' }
        - { path: '/etc/xinetd.conf', mode: '644' }

        path: /etc/ssh/sshd_config

        line: 'PermitRootLogin no'
        state: present
      become: yes

      lineinfile:
        path: /etc/ssh/sshd_config
        regexp: '^Banner'
        line: 'Banner /etc/issue'
        state: present
      become: yes

    - name: Create /etc/issue file
      copy:
        content: |
          *==========================================================================*
          * 경고 : 본 시스템은 인가된 사용자외에는 사용할 수 없습니다. *
          * 불법적인 접근시 관련 법규에 의해 처벌될 수 있습니다. *
          * 문의사항 : 02-2345-6555 시스템 관리자 : 한 재 욱 *

        dest: /etc/issue
        owner: root
        group: root
        mode: '0644'

    - name: Add TMOUT=1800 to /etc/profile
      lineinfile:
        path: /etc/profile

## 4-1. ansible로 구성
```
- hosts: node
  become: yes
  tasks:
    - name: Turn off swap memory
      command: swapoff -a

    - name: Disable swap in /etc/fstab
      lineinfile:
        path: /etc/fstab
        regexp: '^\s*.*\s+swap\s+.*$'
        state: absent

    - name: Disable SELinux
      selinux:
        state: disabled

    - name: Stop and disable firewalld
      systemd:
        name: firewalld
        state: stopped
        enabled: no

    - name: Update dnf.conf to disable SSL verification
      lineinfile:
        path: /etc/dnf/dnf.conf
        regexp: '^sslverify='
        line: 'sslverify=false'

    - name: Install yum-utils package
      yum:
        name: yum-utils
        state: present

    - name: Add Docker repository
      get_url:
        url: https://download.docker.com/linux/centos/docker-ce.repo
        dest: /etc/yum.repos.d/docker-ce.repo

    - name: Install Docker packages
      yum:
        name:
          - docker-ce
          - docker-ce-cli
          - containerd.io
        state: present

    - name: Start and enable Docker service
      systemd:
        name: docker
        state: started
        enabled: yes

    - name: Configure containerd runtime options
      lineinfile:
        path: /etc/containerd/config.toml
        regexp: '^\s*SystemdCgroup\s*=\s*.*$'
        line: 'SystemdCgroup = true'
        create: yes
        backup: yes

    - name: Enable containerd plugin settings
      lineinfile:
        path: /etc/containerd/config.toml
        regexp: '^#?disabled_plugins ='
        line: '#disabled_plugins = ["cri"]'

    - name: Restart and enable containerd service
      systemd:
        name: containerd
        state: restarted
        enabled: yes

```
